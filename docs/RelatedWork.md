You may also want to look at the following projects if you are interested in JNLua:
  * [LuaJava](http://www.keplerproject.org/luajava) - A Script Tool for Java
  * [Lua Scripting Engine for Java](http://www.cynicalmonkey.com/lua-engine/)
  * [kahlua](http://code.google.com/p/kahlua/) - a Lua Virtual Machine for Java
  * [Luaj](http://luaj.sourceforge.net) - a light-weight, Java-centric Lua interpreter
  * [jna-lua](https://github.com/dafrito/jna-lua) - a JSR-223 compliant Lua language binding for Java based on JNA
